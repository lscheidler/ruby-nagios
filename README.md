# Nagios

Provides a basic nagios plugin class for easier plugin development

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'nagios', git: 'https://github.com/lscheidler/ruby-nagios'
```

And then execute:

    $ bundle
